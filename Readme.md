# 🚗 Smart Parking System – HAL Project

## 📌 Project Overview
The Smart Parking System is an intelligent parking management solution developed during my project work at **Hindustan Aeronautics Limited (HAL)**.

This project uses Machine Learning and Computer Vision techniques to detect whether parking slots are occupied or vacant using image data.

The system helps in:
- Reducing time spent searching for parking
- Improving parking space utilization
- Automating manual monitoring processes

---

## 🎯 Objectives
- Detect available and occupied parking slots
- Use a Machine Learning model for classification
- Build a smart solution for real-world parking problems
- Demonstrate practical ML implementation

---

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Jupyter Notebook (.ipynb)

---

## 🧠 Machine Learning Model
- Image-based classification model
- CNN architecture used for prediction
- Model training and testing performed inside Jupyter Notebook
- Complete implementation available in the uploaded `.ipynb` file
